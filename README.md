# Vegan-Bakery-Hut
A website for Vegan Bakery Hut, a conceptual home-based business model focused on supplying and delivering vegan baked goods to local businesses and individual customers. The multi-page website includes the following sections: Home, About, Shop, Reviews, and Order/Contact.

The website was built using HTML5 and CSS3, with a focus on responsive design and accessibility through implementation of Accessible Rich Internet Applications (ARIA) attributes. The development process included the creation of a prototype using high-fidelity wireframes followed by iterative design improvements based upon usability testing, user feedback, research, and surveys.
